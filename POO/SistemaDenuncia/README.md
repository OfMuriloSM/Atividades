# Sistema de Registro de Ocorrências Institucionais

Projeto desenvolvido para a disciplina de Programação Orientada a Objetos — IFCE Campus Canindé.

## Tecnologias utilizadas

- Java
- JavaFX
- FXML
- SQLite
- JDBC
- Maven

## Padrões utilizados

- MVC
- DAO
- Singleton

## Funcionalidades

- Cadastro de ocorrências
- Listagem de ocorrências
- Autenticação do técnico
- Alteração de status da ocorrência

## Como executar

1. Clonar o repositório:
   ```bash
   git clone https://github.com/OfMuriloSM/Atividades/edit/main/POO/SistemaDenuncia.git
   
2. Entrar na pasta do projeto:
   ```bash
    cd SistemaDenuncia
   
3. Executar com Maven:
    ```bash
   mvn clean javafx:run
